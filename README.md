# e-commerce-site
This is a fully responsive e-commerce website frontend built using HTML5, CSS3, and JavaScript. The website includes essential e-commerce UI features such as product listings, product details, shopping cart design, navigation menu, and responsive layout. It is designed to provide a clean and modern user experience for online shopping platforms.

This project demonstrates my ability to build real-world UI for e-commerce platforms and is part of my frontend development portfolio.
